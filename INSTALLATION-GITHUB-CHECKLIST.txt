## ❌ SI ÇA NE MARCHE PAS

### "404 Not Found"
→ Settings → Pages
→ Vérifie: Source = "main", Folder = "/"
→ Attends 10 minutes (première fois)

### "Can't see GitHub Pages option"
→ Crée un fichier vide: docs/index.md
→ Commit
→ Attends 5 min
→ Réessaie

### "AURAGUARDIAN-INFINITI.html affiche du texte (pas de l'app)"
→ Crée _config.yml à la racine:
```yaml
include:
  - ".html"
```
→ Attends 5 min

### "index.html redirect ne marche pas"
→ Remplace le contenu par:
```html
<script>window.location.href = 'AURAGUARDIAN-INFINITI.html';</script>
```
